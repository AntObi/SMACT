def smact.lattice_parameters.b10 (   shannon_radius)

Zn-S-Zn angle is ~109.5 degrees (from a tetrahedron).

It is exactly 2*invCos(-1/3). The distance of that Zn-Zn (diagonally to half the face) is (using the cosine rule) is root[2(r1+r2)^2 - 2(r1+r2)^(2)cos(ZnSZn angle)].
